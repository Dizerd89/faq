# Горячие клавиши
### Копировать: 
Ctrl+C
### Вставить: 
Ctrl+V
### Сохранить:
Ctrl+S
### Вырезать:
Ctrl+X
### Отмена:
Ctrl+Z

# Работа с git
### Скачать проект:
```bash
git clone <url>
```
### Учесть все файлы:
```bash
git add .
```
### сохранить коммит:
```bash
git comnit-m "..."
```
### Отправка в облако:
```bash
git push
```
